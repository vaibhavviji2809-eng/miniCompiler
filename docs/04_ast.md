# AST

The AST represents the program structure after parsing.

Key nodes include:

- `Program`
- `FunctionDeclaration`
- `VariableDeclaration`
- `Assignment`
- `BinaryExpression`
- `UnaryExpression`
- `Literal`
- `Identifier`
- `CallExpression`
- `IfStatement`
- `WhileStatement`
- `ForStatement`
- `ReturnStatement`
